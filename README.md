# the-stock-trader
A fake crypto-stock exchange. User is able to save user data, load user, buy & sell stock.
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```
